# 阿里云服务器配置指南

## 简介

阿里云服务器（Elastic Compute Service，简称ECS）是阿里云提供的一种基础云计算服务，它提供了可伸缩的计算能力，让您可以根据业务需求灵活调整计算资源。

## 购买服务器

1. 登录阿里云官网
2. 选择合适的服务器配置
3. 完成支付流程

## 基础配置

### 连接服务器

```bash
ssh root@your_server_ip
```

### 更新系统

```bash
apt update && apt upgrade -y
```

### 安装必要软件

```bash
apt install -y nginx git nodejs npm
```

## 部署网站

### 配置Nginx

```bash
server {
    listen 80;
    server_name yourdomain.com;
    root /var/www/html;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}
```

### 部署Node.js应用

```bash
git clone your_repository
cd your_repository
npm install
npm run build
pm2 start
```

## 安全配置

1. 设置防火墙
2. 配置SSH密钥登录
3. 禁用密码登录

## 总结

通过以上步骤，您已经完成了阿里云服务器的基本配置和网站部署。更多高级配置和优化，请参考阿里云官方文档。
