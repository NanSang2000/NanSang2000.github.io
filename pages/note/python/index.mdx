# Python 学习笔记

<Callout type="info" emoji="🐍">
Python 是一种简洁、易读、功能强大的编程语言，广泛应用于各个领域。
</Callout>

## 🎯 学习目标

通过系统学习 Python，掌握：

- **语言基础** - Python 语法和数据类型
- **面向对象** - 类、对象、继承等概念
- **标准库** - 常用内置模块和函数
- **实际应用** - Web 开发、数据分析、自动化等

## 🔥 Python 特色

### 语言优势
- **简洁易读** - 语法简单，代码可读性强
- **功能强大** - 丰富的标准库和第三方库
- **跨平台** - 支持 Windows、macOS、Linux
- **应用广泛** - Web、数据科学、AI、自动化等

### 应用领域
- **Web 开发** - Django、Flask 等框架
- **数据科学** - NumPy、Pandas、Matplotlib
- **人工智能** - TensorFlow、PyTorch、scikit-learn
- **自动化** - 脚本编写、运维自动化
- **爬虫** - Scrapy、Requests、BeautifulSoup

## 🛠️ 开发环境

<Callout type="info" emoji="💻">
推荐使用 Python 3.8+ 版本和 PyCharm 或 VS Code 进行开发
</Callout>

### 环境准备
```bash
# 下载安装 Python
https://python.org/downloads/

# 验证安装
python --version
pip --version

# 创建虚拟环境
python -m venv myenv

# 激活虚拟环境
# Windows
myenv\Scripts\activate
# macOS/Linux
source myenv/bin/activate
```

### 包管理
```bash
# 安装包
pip install package_name

# 查看已安装包
pip list

# 生成依赖文件
pip freeze > requirements.txt

# 安装依赖
pip install -r requirements.txt
```

## 📚 学习路径

### 基础篇
- **变量和数据类型** - 数字、字符串、列表、字典等
- **控制流程** - 条件语句、循环语句
- **函数定义** - 函数参数、返回值、作用域
- **文件操作** - 文件读写、异常处理

### 进阶篇
- **面向对象** - 类的定义、继承、多态
- **模块和包** - 模块导入、包的创建
- **装饰器** - 函数装饰器、类装饰器
- **生成器** - yield 关键字、迭代器

### 实战篇
- **Web 开发** - Flask/Django 框架
- **数据处理** - Pandas 数据分析
- **网络编程** - 爬虫、API 开发
- **自动化脚本** - 系统管理、任务自动化

## 🎉 开始学习

Python 是一门非常适合初学者的编程语言，同时也能满足专业开发的需求。

### 学习建议
1. **多练习** - Python 重在实践，多写代码才能熟练掌握
2. **读源码** - 阅读优秀的 Python 项目源码
3. **做项目** - 通过实际项目来应用所学知识
4. **关注社区** - 参与 Python 社区讨论和开源项目

<Callout type="warning" emoji="⚠️">
Python 有很多版本，建议使用 Python 3.8 或更高版本进行学习。
</Callout>
