## 设计特点

新的访客卡片设计包含以下特点：

- 🎨 **现代化渐变设计** - 使用 emerald 到 cyan 的渐变色彩
- ✨ **丰富的动画效果** - 包括入场动画、悬停动画和旋转效果
- 🌟 **浮动粒子背景** - 动态的粒子装饰增加视觉吸引力
- 📱 **响应式设计** - 在不同屏幕尺寸下都有良好的显示效果
- 🌙 **深色模式支持** - 完整的深色模式适配
- 💡 **状态指示器** - 实时状态显示和加载动画
- 🔢 **数字动画** - 平滑的数字增长动画效果
- 📍 **IP地址显示** - 美观的IP信息展示区域

## 使用方式

```jsx
import VisitorCard, { VisitorCardMini } from '../components/Visitor/VisitorCard'

// 基础使用
<VisitorCard />

// 自定义标题
<VisitorCard titleText="自定义标题" />

// 隐藏标题
<VisitorCard showTitle={false} />

// 禁用动画
<VisitorCard showAnimation={false} />

// 紧凑版本
<VisitorCardMini showTitle={true} titleText="访客统计" />
```
